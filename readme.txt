Create custom fields for any post types. Friendly interface built by Reactjs.

== Description ==
You can add more custom fields type:
* Input type text, password, color, date, datetime local, email, month, number, time, url, week
* Multiple Input
* Text Area
* Checkbox
* Radio
* Select
* File
* Image
* WP Editor

If you have any issues, feel free to contact me: https://github.com/Viet27th/toolazy-custom-fields/issues

== Frequently Asked Questions ==

= How to use it? =

After installed and active this plugin, look at to Admin menu on WP admin page, make sure that you are logged in admin account,
click to "Toolazy CF -> Introduce page -> Change default Permalinks". After that, go to "Toolazy CF -> Setting page", click to expand the Post Type you want to add a new Custom Fields.
After create a new custom field, it should be appear on the Create screen of that Post Type
